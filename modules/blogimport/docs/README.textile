h2. About

The Tumblr Import module imports your tumblr blog into PyroCMS. 

h2. Usage

# Upload the tumblrimport directory to addons/modules
# Log into the admin, and enable the add-on at htpp://www.yourwebsite.com/admin/modules
# Go to http://www.yourwebsite.com/admin/tumblrimport
